### 开始学习

此仓库为学习文档的配套示例源码，按照下面的命令获取仓库：

```shell
git clone https://github.com/100askTeam/esp-idf-100ask-learn.git
git submodule update --init --recursive
```

将仓库更新到最新状态：
```shell
cd esp-idf-100ask-learn  # 先进入到 esp-idf-100ask-learn 目录
git submodule update --init --recursive
```
